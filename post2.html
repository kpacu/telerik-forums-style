Решение: html, sass, demo.

В началото съм сложил няколко променливи. Стилизирам в следния ред: reset, тагове, списъци, grid, кутийки, форми, меню и накрая специфичните за сайта елементи. HTML файлът, беше добре написан, но му направих малки промени. Може да се изнесат и още неща като променливи, но не се повтаряха и ги оставих така засега.

С познатия клас .inline от миналите домашни, менюто много лесно се сменя от вертикално на хоризонтално:

.inline {
    @extend .clearfix
    li { float: left }
}
 
.menu {
    a { color: #fff; }
    ...
}
 
.menu-horizontal {
    @extend .inline
    @extend .menu
}
 

Малко по-дълго стана, но пък си проличава предимството на предпроцесорите. Тези класове ще могат лесно да се преизползват и ще стане по-бързо писането на следващите домашни.

Edit: Малки промени по кода.
